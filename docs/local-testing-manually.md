# local testing

## install 8.0.0

```bash
asdf uninstall php 8.0.0
rm -rf ~/.asdf/plugins/php
ln -s /home/dev/Documents/asdf-community/asdf-php ~/.asdf/plugins/php
asdf install php 8.0.0
```
